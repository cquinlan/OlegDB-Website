Introducing: Project Oleg

OlegDB is a database that meets the bottom line head-on.

  • Big Data
  • Small Data
  • Clustering with Bash
  • ACID compliance

Written and designed with battle-tested concepts

The 1980's were a glorious time for database experimentation, with the clear winner being DBM. With that in mind, we built OlegDB.

  • Written in pure ANSI C
  • Hard keysize limit of 16 characters
  • Single-threaded
  • Is RESTful
  • Respects your Freedom

Portable by most definitions of the word

OlegDB exports an API, so if you want to embed it go ahead. Don't want to worry about it? Use the server!
